# Deno Examples to get started.

## Installing Deno

### On Mac os

```
  brew install deno
```

### Linux
```
  curl -fsSL https://deno.land/x/install/install.sh | sh
```

### Windows
```
  choco install deno
```
#### For Installing chocolately in windows https://chocolatey.org/docs/installation
